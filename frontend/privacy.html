개인정보 처리방침

시행일: 2026-05-23

1. 수집하지 않는 정보

NewsLens는 다음 정보를 수집·저장하지 않습니다.

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 회원 가입 정보, 비밀번호
  • 분석에 입력하신 기사의 원문 URL (sha256 해시값만 보관)
  • 분석에 입력하신 기사의 본문 텍스트
  • 분석 결과(JSON)는 24시간만 메모리 캐시, 영구 저장하지 않음

2. 일시적으로 사용하는 정보

서비스 운영을 위해 다음 정보를 일시적으로만 사용합니다.

  • 접속 IP 주소: 요청 빈도 제한(시간당 20회) 검증 용도. 메모리에서 1시간 슬라이딩 윈도우 후 자동 폐기. 디스크 저장 없음.
  • 입력한 기사 URL: 24시간 결과 캐시 키(sha256)로 사용. 원문 URL은 캐시 만료 후 즉시 삭제.

3. 피드백 수집 시 저장하는 정보

사용자가 분류 결과에 대한 피드백(👍/👎/카테고리 제안)을 보낼 때만 다음을 저장합니다.

  • 기사 URL의 sha256 해시 (역추적 불가능한 식별자)
  • 문장 번호 (몇 번째 문장에 대한 피드백인지)
  • 원래 분류 카테고리 / 사용자 제안 카테고리
  • 피드백 유형 (thumbs_up / thumbs_down / category_correction)
  • UTC 타임스탬프

원문 URL, 기사 본문, 문장 텍스트, 사용자 식별 정보는 일체 저장되지 않습니다.

4. 외부 서비스 이용

분석을 위해 다음 외부 서비스에 기사 본문이 전송됩니다.

  • Google Gemini API (https://ai.google.dev) — 사실/의견 분류 및 핵심 사실 요약을 위해 본문이 전송됩니다. Google의 API 데이터 사용 정책은 해당 약관을 따릅니다.

5. 쿠키

NewsLens는 추적·광고용 쿠키를 사용하지 않습니다. 브라우저의 localStorage는 이 페이지의 안내 배너를 한 번 본 뒤 다시 표시하지 않기 위한 cookie_notice_dismissed 키 하나만 사용합니다.

6. 결과의 한계

본 분석은 AI 모델 기반 참고 자료이며, 전문적 미디어 비평이나 사실 검증을 대체하지 않습니다. 최종 판단은 사용자에게 있습니다.
